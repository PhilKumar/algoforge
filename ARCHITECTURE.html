🚀 Zero-Downtime Blue-Green Deploy
1
Push to main triggers pipeline
ci.yml → playwright.yml → deploy.yml (sequential, each gates next)
2
SSH into EC2, pull latest code
git fetch origin main · git reset --hard · pip install -q -r requirements.txt
3
Read active port, start standby
~/.algoforge-active-port → systemctl start algoforge@{standby-port}
4
Health-check standby (30s timeout)
curl /api/health every 1s · rollback if timeout: stop standby, abort deploy
5
Rewrite Nginx upstream & reload
echo upstream → tee conf · nginx -t · nginx -s reload (graceful, 0 dropped reqs)
6
Drain old WebSocket connections
sleep 30s — active sessions finish naturally on old uvicorn worker
7
Stop old instance · save port state
systemctl stop algoforge@{old} · echo new-port > ~/.algoforge-active-port
🛡️ Error Recovery Matrix
Failure Severity Detection Recovery Action
Deploy health check fails CRITICAL curl /api/health timeout 30s Standby stopped · active port file unchanged · CI job fails · Nginx untouched
Dhan JWT token expired CRITICAL JWT decode on startup · days_left ≤ 0 TOTP auto-regeneration via pyotp · new token written to .current_token · fcntl lock prevents race
Process crash CRITICAL systemd watchdog Restart=always · RestartSec=5 · atexit saves all engine states to runs.json before exit
WebSocket disconnect HIGH on_close callback Auto-reconnect with exponential backoff: 1s → 2s → 4s → 8s → 16s → 32s cap
429 Rate Limit HIGH HTTP 429 status code Read Retry-After header · fallback to 60s · exponential backoff on subsequent hits
API 5xx error HIGH HTTP 5xx status Retry 3× with backoff · Telegram alert on exhaustion · engine pauses until next candle
Invalid HMAC signature (Delta) MEDIUM HTTP 401 response Regenerate timestamp · retry once · log discrepancy if retry also fails
Historical candle data gap MEDIUM Gap detector in data_downloader.py Targeted re-fetch for missing range · Binance public API fallback if Delta returns empty
INR rate API unreachable MEDIUM fetch() exception in browser JS Fallback to hardcoded ₹84.5/USD · silent fail · retry on next 15-minute refresh cycle
